getPersonMatches
functions in module searchAndMatch

Description

Get the matches (possible duplicates) for a person. Set the collection query parameter to https://familysearch.org/platform/collections/records to get record hints.

The response includes the following convenience functions

  • getSearchResults() - get the array of SearchResults from the response
  • getResultsCount() - get the total number of search results
  • getIndex() - get the starting index of the results array
