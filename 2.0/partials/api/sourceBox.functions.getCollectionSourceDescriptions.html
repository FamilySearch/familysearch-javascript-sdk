getCollectionSourceDescriptions
functions in module sourceBox

Description

Get a paged list of source descriptions in a user-defined collection The response includes the following convenience function

  • getSourceDescriptions() - get an array of SourceDescriptions from the response

FamilySearch API Docs

Usage

getCollectionSourceDescriptions(url[, params]);

Parameters

ParamTypeDetails
urlString

full URL of the collection-source-descriptions endpoint

params
(optional)
Object

count maximum to return (defaults to 25), start zero-based index of first source to return
