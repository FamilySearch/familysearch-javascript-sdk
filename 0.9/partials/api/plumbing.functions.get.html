get
functions in module plumbing

Description

Low-level call to get a specific REST endpoint from FamilySearch

Usage

get(url[, params][, headers][, opts][, responseMapper]);

Parameters

ParamTypeDetails
urlString

may be relative; e.g., /platform/users/current

params
(optional)
Object

query parameters

headers
(optional)
Object

options headers

opts
(optional)
Object

options to pass to the http function specified during init

responseMapper
(optional)
Function

function to map response data to something else

Returns

Object

a promise that behaves like promises returned by the http function specified during init
