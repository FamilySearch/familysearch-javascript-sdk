getAccessTokenWithClientCredentials
functions in module authentication

Description

Get an access token via client credentials.

Usage

getAccessTokenWithClientCredentials(key[, password][, time]);

Parameters

ParamTypeDetails
keyString

A PEM-encoded key. Matches the pem parameter of ursa.createPrivateKey().

password
(optional)
String

The password for descrypting the key, if necessary.

time
(optional)
IntegerString

This parameter is only available for the sake of testing. Do not use it in production.
