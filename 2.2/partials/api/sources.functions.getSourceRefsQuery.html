getSourceRefsQuery
functions in module sources

Description

Get the people, couples, and child-and-parents relationships referencing a source The response includes the following convenience functions

  • getPersonSourceRefs() - get an array of person SourceRefs from the response
  • getCoupleSourceRefs() - get an array of couple relationship SourceRefs from the response
  • getChildAndParentsSourceRefs() - get an array of child and parent relationship SourceRefs from the response
