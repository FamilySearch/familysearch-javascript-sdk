getCollectionSourceDescriptionsForUser (functions in module sourceBox )

Description

Get a paged list of source descriptions in all user-defined collections defined by a user The response includes the following convenience function

  • getSourceDescriptions() - get the array of source descriptions from the response; each has the same convenience functions as for getCollectionSourceDescriptions
