http (functions in module plumbing )

Description

Low-level call to issue an http request to a specific REST endpoint from FamilySearch

Usage

http(method, url[, headers][, data][, opts][, responseDataExtender][, retries]);

Parameters

  • method – {String} –

    GET, POST, PUT, or DELETE

  • url – {String} –

    may be relative

  • headers(optional) – {Object=} –

    optional headers object

  • data(optional) – {Object=} –

    optional post data

  • opts(optional) – {Object=} –

    optional options to pass to the http function specified during init

  • responseDataExtender(optional) – {Function=} –

    optional function to extend response data

  • retries(optional) – {Number=} –

    optional number of times to retry

Returns

{Object} –

a promise that behaves like promises returned by the http function specified during init
