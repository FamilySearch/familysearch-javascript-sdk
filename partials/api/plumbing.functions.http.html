http (functions in module plumbing )

Description

Low-level call to issue an http request to a specific REST endpoint from FamilySearch

Usage

http(method, url[, headers][, data][, opts][, responseMapper][, retries]);

Parameters

  • method – {String} –

    GET, POST, PUT, or DELETE

  • url – {String} –

    may be relative

  • headers(optional) – {Object=} –

    headers object

  • data(optional) – {Object=} –

    post data

  • opts(optional) – {Object=} –

    options to pass to the http function specified during init

  • responseMapper(optional) – {Function=} –

    function to map response data into the data to return

  • retries(optional) – {Number=} –

    number of times to retry

Returns

{Object} –

a promise that behaves like promises returned by the http function specified during init
