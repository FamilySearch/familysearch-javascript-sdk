getAuthCode
functions in module authentication

Description

Either Open a popup window to allow the user to authenticate and authorize this application, or redirect the user to the FamilySearch authentication endpoint with the current URL as a state that will redirect on success. You do not have to call this function. If you call getAccessToken without passing in an authorization code, that function will call this function to get one.

Usage

getAuthCode();

Returns

Object

a promise of the (string) auth code
