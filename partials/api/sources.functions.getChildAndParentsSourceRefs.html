getChildAndParentsSourceRefs
functions in module sources

Description

Get the source references for a child and parents relationship The response includes the following convenience function

  • getSourceRefs() - get an array of SourceRefs from the response
  • getSourceDescriptions() get an array of SourceDescriptions from the response
  • getSourceDescription(id) get the SourceDescription with the specified source description id from the response
