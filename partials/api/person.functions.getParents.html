getParents
function in module person

Description

Get the relationships to a person's parents. The response includes the following convenience functions

  • getChildAndParentsRelationships() - an array of ChildAndParents relationships
  • getCoupleRelationships() - an array of Couple relationships for parents
  • getPerson(pid) - a Person for any person id in a relationship
