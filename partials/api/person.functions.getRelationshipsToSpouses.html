getRelationshipsToSpouses
functions in module person

Description

Get the relationships to a person's spouses. The response includes the following convenience functions

  • getSpouseIds() - an array of string ids
  • getRelationships() - an array of Couple relationships
  • getPerson(pid) - if the persons parameter has been set, this function will return a Person for a person id in the relationship
