getChildAndParents
functions in module parentsAndChildren

Description

Get information about a child and parents relationship. The response includes the following convenience functions

  • getRelationship() - a ChildAndParents relationship
  • getPerson(pid) - if the persons parameter has been set, this function will return a Person for a person id in the relationship
