DiscussionRef
type in module discussions

Description

Reference to a discussion on a person. To create a new discussion reference, you must set $personId and discussion. NOTE: discussion references cannot be updated. They can only be created or deleted.

Usage

DiscussionRef([data]);

Parameters

ParamTypeDetails
data
(optional)
Object

an object with optional attributes {$personId, discussion} discussion can be a Discussion or a discussion URL or a discussion id
