getPlaceTypeGroup
functions in module places

Description

Get a Place Type Group which includes a list of Places Types in the group.

  • getList() - get the VocabularyList (Place Type Group) from the response
  • getPlaceTypes() - get an array of the VocabularyElements (Place Types) in the group
