getPlacesSearch
functions in module places

Description

Search for a place.

  • getSearchResults() - get an array of PlacesSearchResults from the response.

Search Parameters

  • start - The index of the first search result for this page of results.
  • count - The number of search results per page.
  • name
  • partialName
  • date
  • typeId
  • typeGroupId
  • parentId
  • latitude
  • longitude
  • distance

Read the API Docs for more details on how to use the parameters.

Usage

getPlacesSearch(params, lang);

Parameters

ParamTypeDetails
paramsObject

Search parameters

langString

Optional. The language you want the display data returned in, when available. In the browser this defaults to the browser's language. On the server this defaults to 'en'.

Returns

Object

promise for the response
