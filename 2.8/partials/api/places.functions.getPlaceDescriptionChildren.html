getPlaceDescriptionChildren
functions in module places

Description

Get the children of a Place Description. Use getPlacesSearch() to filter by type, date, and more.

  • getChildren() - get an array of the PlaceDescriptions (children) from the response

API Docs

Usage

getPlaceDescriptionChildren(url, lang);

Parameters

ParamTypeDetails
urlString

full url for the place descriptions children endpoint

langString

Optional. The language you want the display data returned in, when available. In the browser this defaults to the browser's language. On the server this defaults to 'en'.

Returns

Object

promise for the response
