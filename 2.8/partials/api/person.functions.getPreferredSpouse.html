getPreferredSpouse
functions in module person

Description

Get the preferred Couple relationship id if any for this person and this user. The response has the following convenience function:

  • getPreferredSpouse() - returns the url of the preferred couple relationship, null if it is the unknown spouse, or undefined if there is no preference
